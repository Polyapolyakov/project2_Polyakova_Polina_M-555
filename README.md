# project2_Polyakova_Polina_M-555
Примитивная база данных

# описание:
Адаптированное консольное приложение, имитирующее работу с базой данных: создание, наполнение, обновление и удаление таблиц и записей в них, а также построение пользовательских запросов с соответсвующей визуализацией результата работы запроса.

# установка зависимостей:
make install

# запуск приложения:
make project

# помощь:
Для упрощения работы с прирожением и его функционалом для работы пользователя предусмотрена соответсвующая навигация, оптимизирующая работу с приложением (пользователю доступны не только перечень команд и их описание, но и унифицированная форма записи для целей устранения ошибок ввода данных).
